This section provides interactive endpoint documentation generated from the OpenAPI specification.

## Authentication

All endpoints require the `X-API-Key` header. Set your key in the playground to try requests.

## Base URL

```
https://<your-instance>/api/v2
```

Replace `<your-instance>` with your BuildingSwell host (for example, `app.buildingswell.com`).

## Standard CRUD

Most resources follow the same endpoint pattern documented in [standard endpoints](/guides/standard-endpoints). This reference lists the concrete paths for each resource group.

For querying, filtering, and joins, see the [querying guide](/guides/querying).
