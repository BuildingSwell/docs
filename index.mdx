Everything you need to integrate with the BuildingSwell v2 API.

## Base URL

```text
https://app.buildingswell.com/api/v2
```

All responses are `Content-Type: application/json`, except multipart for Excel imports.

## Quick start

<Steps>
  <Step title="Get an API key">
    Create an API key in BuildingSwell (requires admin access). The `secretKey` is returned exactly once on creation — store it immediately.
  </Step>
  <Step title="Authenticate requests">
    Include your API key in every request:

    ```bash
    curl "https://app.buildingswell.com/api/v2/project" \
      -H "X-API-Key: <your-api-key>"
    ```
  </Step>
  <Step title="Explore resources">
    Every resource supports a standard CRUD pattern. Start with [standard endpoints](/guides/standard-endpoints) or jump to a specific [resource](/resources/project).
  </Step>
</Steps>
